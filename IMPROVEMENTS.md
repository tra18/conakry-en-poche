## 🔧 **Installation et Déploiement**

### 📋 **Prérequis**
```bash
# Node.js 16+
# npm ou yarn
# Firebase CLI
```

### 🚀 **Installation**
```bash
# Cloner le projet
git clone https://github.com/your-repo/conakry-en-poche.git

# Installer les dépendances
npm install

# Configurer les variables d'environnement
cp .env.example .env

# Démarrer en développement
npm start

# Build pour production
npm run build
```

### 🌐 **Déploiement**
```bash
# Firebase Hosting
firebase deploy

# Vercel
vercel deploy

# Netlify
netlify deploy --prod
```
